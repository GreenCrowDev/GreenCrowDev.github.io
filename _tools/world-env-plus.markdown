**World Env Plus** is an addon for the Godot Engine that provides 3D environment utilities.
It implements environment volumes and allows environment changes when entering areas, similar to what other engines provide out of the box.

![Screenshot](/assets/tools/world-env-plus/img/world_env_plus_001.png)

The addon has two main goals:

1. Provide better editor controls to quickly iterate on environment settings and visuals.
2. Implement volumes that change the environment when entering specific areas.

![Screenshot](/assets/tools/world-env-plus/img/world_env_plus_002.png)

The tool is under active development in collaboration with [FR3NKD](https://x.com/FR3NKD).

We plan to publish it on the Godot Asset Store as soon as a stable release is ready.
